# Lionael Surya — Portfolio

Premium interactive portfolio showcasing data engineering, analytics engineering, and software engineering work.

**Built like production:** tested, documented, reproducible, and shipped.

## Tech Stack

- **Framework:** Next.js 16.2.10 (App Router)
- **Language:** TypeScript (strict mode)
- **Styling:** Tailwind CSS 4
- **Animations:** Motion (Framer Motion)
- **Fonts:** Geist Sans & Geist Mono
- **Deployment:** Vercel

## Features

- Server Components by default for optimal performance
- Interactive case study modals with architecture explorer
- Custom OpenGraph images
- JSON-LD structured data
- Full keyboard navigation & screen reader support
- Reduced motion support
- WCAG AA compliant
- SEO optimized (sitemap, robots, meta tags)
- Responsive: 320px → ultrawide
